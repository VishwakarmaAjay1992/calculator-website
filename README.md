# Financial Calculator Website

A free and easy-to-use financial calculator website designed to help users calculate investments, loans, retirement planning, deposits, and government savings schemes.

The website is built using HTML, CSS, and JavaScript and is hosted using GitHub Pages.

## Live Website

Visit the website here:

https://vishwakarmaajay1992.github.io/calculator-website/

## Available Calculators

The website currently includes the following calculators:

- SIP Calculator
- SWP Calculator
- Loan EMI Calculator
- Deposit Calculator
- Retirement Calculator
- Government Schemes Calculator

## Main Features

- Simple and user-friendly interface
- Responsive design for desktop and mobile devices
- Instant calculation results
- Investment growth calculations
- Loan EMI calculations
- Retirement planning estimates
- Government savings scheme calculations
- Detailed result summaries
- Financial planning information
- No registration required
- Free to use

## Technologies Used

- HTML5
- CSS3
- JavaScript
- GitHub Pages
